# PS1 Classic Pi Lite

Uma distribuição leve baseada no Raspberry Pi OS Lite dedicada exclusivamente ao PlayStation.

## Objetivos

- Raspberry Pi 3
- Boot rápido
- Apenas PlayStation
- Instalação modular
- Código aberto
